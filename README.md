## 🎨 О ДИЗАЙНЕ

### Концепция: "Мягкая Роскошь"

Премиальный дизайн для психологического сервиса с использованием пастельной палитры:

**Цветовая палитра:**
- 💜 **Лавандовый** (#e5d4fa) - спокойствие, духовность
- 💙 **Голубой** (#b6d1f2) - доверие, ясность
- 💗 **Розовый** (#ffe0e1) - забота, эмпатия
- 🌟 **Кремовый** (#fff9eb) - уют, тепло
- 🩵 **Барвинковый** (#a0aed6) - мудрость, стабильность

**Типографика:**
- **Заголовки:** Playfair Display (serif) - роскошь
- **Текст:** Inter (sans-serif) - читаемость

**Ключевые особенности:**
- ✨ Огромные отступы (160px между секциями)
- ✨ Мягкие тени (opacity 0.03-0.08)
- ✨ Glassmorphism эффекты
- ✨ Цветные градиенты
- ✨ Премиальная типографика

## ⚙️ Настройка перед публикацией

### 1. Аналитика (2 минуты):

Откройте `js/analytics.js` и замените:

```javascript
const GA_MEASUREMENT_ID = 'G-XXXXXXXXXX'; // ← Ваш Google Analytics 4 ID
const YM_COUNTER_ID = null; // ← Ваш Yandex Metrika ID
```

### 2. Изображения (опционально, 10 минут):

Создайте в `images/`:
- `logo.png` (512x512px, лавандовый градиент)
- `og-image.jpg` (1200x630px)
- `favicon.ico`
- `icons/icon-*.png` (72-512px)

### 3. Ссылки (5 минут):

- Обновите ссылки на App Store/Google Play в секции #download
- Проверьте социальные сети в футере
- Обновите email адреса
